title: Moc–数据库设计

前言

在本章中主要介绍新建Django项目，以及项目的配置和数据库设计

创建项目

创建项目的两种方式

  • 命令行创建
    在前些文章中，讲述过通过命令行去创建Django项目，在此不再赘述
  • 通过pycharm创建
    • 打开pycharm，选择create New Project 或者 File -> New Project
    • 选择项目路径，选择项目的环境

配置Django项目

数据库配置

  • 打开项目根app下的settings.py,找到DATABASES
  • 配置项为: 引擎，数据库名，用户名，密码，以及主机

Navicat 创建数据库

MySQL的安装，可以参见我的博客,Navicat的安装使用自行Google

  • 打开 Navicat，新建数据链接，名字任意，用户密码主机等等要与刚才配置的保持一致

  • 新建数据库，命名为mxonline

  • 数据迁移
    本项目中我们使用pycharm自带的终端系统进行数据迁移以及其他的终端操作
    Tools ->Run manage.py task
    然后分别执行数据迁移命令：
    makemigrations
    migrate

We will mainly talk about that how can we create the Django project and the design database in this chapter. There are 4 main apps in our project: users,course，organization，operation。The reason why we design like this is that we will discuss later.
